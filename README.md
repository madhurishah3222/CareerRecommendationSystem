# 🚀 Career Recommendation System

A Machine Learning-based web application that analyzes student academic performance and interests to recommend suitable career paths.

---

## 📌 Features
- 🎯 Personalized career recommendations based on user input
- 📊 Uses Machine Learning (Scikit-learn) for prediction
- 🧠 Analyzes academic scores and interests
- 🌐 Built with Django backend
- 📋 Simple and user-friendly interface

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Django (Python)
- **Machine Learning:** Scikit-learn
- **Database:** SQLite

---

## ⚙️ How It Works
1. User enters academic details and interests  
2. Data is processed using a trained ML model  
3. System predicts and recommends suitable career paths
